# avianjets-rocket

Takes distance as input and calculates travel time with fibonacci loop.

## Installation

Download or clone the repository.

```sh
npm install
npm run build
```
## Run from CLI

```sh
cd ./build/bin
node rocket 384400
```

## Tests

```sh
npm install
npm test
```

## Dependencies

None
